# ADR 数值精度分析与改进计划

> 生成日期: 2026-05-07
> 基于 demo3 代码 (solver.cpp, runtime.cpp, io.cpp) 分析

---

## 1. PDE 系统概述

当前求解的无量纲化 bulk-boundary coupled 系统:

**Bulk (2D domain, x in [0, L], y in [0, 1]):**

```
dc/dt + u(x,y,t) * dc/dx = Laplacian(c)
```

其中速度场只有 x 分量:

```
u(y,t) = Pe * y(1-y) + Pe2 * f(y,t)
```

`f(y,t)` 是振荡流分量 (Womersley 型, 频率 w = 2*alpha^2*Sc).

**Boundary (y = 0, Langmuir adsorption):**

```
d(eta)/dt = eps*Da * [ c|_{y=0} * (1-eta) - eta/K0 ]
```

**Coupling (Robin BC at y = 0):**

```
dc/dy|_{y=0} = Da * [ c|_{y=0} * (1-eta) - eta/K0 ]
```

**其他边界条件:**
- x = 0: c = c0 = 1 (Dirichlet, 入口)
- x = L: dc/dx = 0 (Neumann, 出口)
- y = 1: dc/dy = 0 (Neumann, 上壁)

## 2. 实际计算参数范围

| 参数 | 符号 | 最小值 | 最大值 | 含义 |
|------|------|--------|--------|------|
| Peclet (稳态流) | Pe | 10 | 10^5 | 对流/扩散比 |
| Peclet (振荡流) | Pe2 | 10^(-2) | 10^6 | 振荡对流强度 |
| Womersley 数 | alpha | 0.001 | 10 | 振荡频率参数 |
| Damkoehler 数 | Da | 100 | 100 | 吸附/扩散比 |
| 表面容量参数 | eps | 0.1 | 0.1 | 表面-体相耦合强度 |
| 平衡常数 | K0 | 1.0 | 1.0 | 吸附平衡 |
| y 方向网格数 | ny | 50 | 100 | 网格分辨率 |
| Schmidt 数 | Sc | 16667 | 16667 | 动量/质量扩散比 |
| 波长参数 | lam | 0.1 | 0.1 | 决定 x 方向域长 |

**导出量:**

```
h = 1/ny                         (网格间距, 0.01 ~ 0.02)
nx = 1/(lam*h)                   (x 方向网格数)
u_max ~ Pe/4 + Pe2*|f|_max       (最大速度)
w = 2*alpha^2*Sc                 (振荡角频率)
T_osc = pi/(alpha^2*Sc)          (振荡周期)
```

## 3. 空间离散精度分析

### 3.1 当前方案: 扩散中心差分 + 对流一阶 upwind

**扩散项 (calc_phi):** 标准二阶中心差分, 截断误差 O(h^2). 在所有参数范围内无问题.

**对流项 (advection_c):** 一阶 upwind 差分. 修正方程为:

```
dc/dt + u*dc/dx = (1 + |u|*h/2) * d^2c/dx^2 + O(h^2)
```

数值扩散系数 = |u|*h/2, 物理扩散系数 = 1.

### 3.2 各参数区间的数值扩散估算

以 y = 0.5 处 Poiseuille 分量最大值 u_Poiseuille = Pe/4 为代表:

| Pe | h = 0.02 | h = 0.01 | 物理扩散 | 扩散比 (h=0.02) |
|----|----------|----------|----------|-----------------|
| 10 | 0.025 | 0.0125 | 1 | **2.5%** OK |
| 10^2 | 0.25 | 0.125 | 1 | **25%** 偏高 |
| 10^3 | 2.5 | 1.25 | 1 | **250%** 不可用 |
| 10^4 | 25 | 12.5 | 1 | **2500%** 不可用 |
| 10^5 | 250 | 125 | 1 | **25000%** 不可用 |

**Pe2 振荡分量的数值扩散:**

振荡流最大速度 |f|_max ~ O(1) (归一化), 所以振荡对流项贡献的数值扩散 ~ Pe2*|f|_max*h/2.

| Pe2 | 数值扩散 (h=0.02) | 扩散比 |
|-----|-------------------|--------|
| 10^(-2) | 10^(-4) | 0.01% OK |
| 1 | 0.01 | 1% OK |
| 10^2 | 1 | 100% 不可用 |
| 10^4 | 100 | 10000% 不可用 |
| 10^6 | 10000 | 10^6% 不可用 |

### 3.3 结论: 空间精度可靠区间

**仅 Poiseuille 流 (Pe2 可忽略):**
- Pe <= 50: 数值扩散 < 5%, **结果可靠**
- 50 < Pe <= 200: 数值扩散 5-25%, **定性可信, 定量有偏差**
- Pe > 200: 数值扩散 > 物理扩散, **结果由数值伪项主导**

**含振荡流:**
- Pe2 <= 50: 同上
- Pe2 > 100: 振荡流方向的数值扩散也超过物理扩散

**你的参数范围内, Pe = 10~10^5, Pe2 = 0.01~10^6, 大部分工况落在不可用区间.**

## 4. 时间离散精度分析

### 4.1 当前方案: 全显式 Forward Euler

**CFL 约束:**

```
dt <= min(dt_diffusion, dt_advection)

dt_diffusion = h^2 / (2*dim) = h^2/4             (2D)
dt_advection = h / |u_max|
```

| Pe | h | dt_diffusion | dt_advection | 约束瓶颈 |
|----|---|-------------|-------------|----------|
| 10 | 0.02 | 10^(-4) | 2e-3 | 扩散 |
| 10^2 | 0.02 | 10^(-4) | 8e-4 | 扩散 |
| 10^3 | 0.02 | 10^(-4) | 8e-5 | **对流** |
| 10^4 | 0.02 | 10^(-4) | 8e-6 | **对流** |
| 10^5 | 0.02 | 10^(-4) | 8e-7 | **对流** |

### 4.2 eta 的额外约束

当前显式 Euler 对 eta 的稳定性要求:

```
dt < 1 / (eps*Da*(c + 1/K0))
```

对 eps*Da = 10, c = 1, K0 = 1: dt < 1/20 = 0.05.
当前参数下这不是瓶颈, 但如果 Da 增大到 1000 则可能成为约束.

### 4.3 振荡流精度约束

振荡周期 T_osc = pi/(alpha^2 * Sc):

| alpha | T_osc | 精度要求 dt < T/20 |
|-------|-------|-------------------|
| 0.001 | pi/(10^(-6)*16667) ~ 188 | 9.4 |
| 0.01 | pi/(10^(-4)*16667) ~ 1.88 | 0.094 |
| 0.1 | pi/(0.01*16667) ~ 0.019 | 9.4e-4 |
| 1 | pi/(1*16667) ~ 1.88e-4 | 9.4e-6 |
| 10 | pi/(100*16667) ~ 1.88e-6 | 9.4e-8 |

alpha 大时, 振荡频率极高, 时间步要求极小 -- 这解释了 "alpha 大时 dt_limit 和 Pe 无关".

### 4.4 综合时间步约束总结

```
dt_actual = min(h^2/4,  h/u_max,  T_osc/20,  1/(eps*Da*2))
```

| 区域 | 主导约束 | 隐式化能否放松 |
|------|---------|--------------|
| alpha > 1 | 振荡精度 T_osc/20 | **不能** (精度约束, 非稳定性) |
| alpha < 0.1, Pe < 100 | 扩散 CFL h^2/4 | 扩散隐式可放松 |
| alpha < 0.1, Pe > 1000 | 对流 CFL h/u_max | 必须处理对流项 |
| alpha ~ 0.1 | 三者竞争 | 视具体参数 |

## 5. 影响矩阵: 参数组合 vs 需要的改进

```
                    空间精度足够        空间精度不足
                    (Pe,Pe2 < 50)      (Pe 或 Pe2 > 200)
                 +------------------+----------------------+
  alpha > 1      | 现状可用           | 需修空间离散           |
  (振荡主导)     | dt 受振荡限制      | dt 受振荡限制          |
                 | 隐式化无收益       | 时间积分无需改          |
                 +------------------+----------------------+
  alpha < 0.1    | 需修时间积分        | 两者都需要修           |
  Pe < 100       | 扩散隐式即可        | 空间为主, 时间为辅      |
                 +------------------+----------------------+
  alpha < 0.1    | 需修时间积分        | 两者都需要修           |
  Pe > 1000      | 必须含对流隐式      | **最高优先级**         |
                 +------------------+----------------------+
```

**你的参数范围 (Pe 最高 10^5, Pe2 最高 10^6, alpha 最低 0.001) 意味着右下角是核心工况.**

## 6. 改进方案

### Phase 0: 验证基线 (在动手修改之前)

**目标:** 建立"当前代码在哪些参数下结果正确"的基准线.

1. 选取 Pe = 10, alpha = 0.1 的 case, 与已知解析解 / 文献对比.
2. 同一 case 在 ny=50 和 ny=100 下跑, 检查网格收敛性 (结果应接近).
3. 选取 Pe = 1000 的 case 同样做网格收敛性检查 -- 预期 ny=50 和 ny=100 差异很大, 确认数值扩散问题.

### Phase 1: 空间精度升级 (优先级最高)

**目标:** 消除高 Pe 下的数值扩散, 使 Pe = 10^5 的结果可靠.

**方案: 替换 advection_c 中的一阶 upwind 为高精度格式**

推荐路线 (从简到繁, 逐级验证):

#### 1a. 二阶 upwind (最小改动)

```cpp
// u > 0 时:
adv = u * (3*cc[i] - 4*cc[i-1] + cc[i-2]) / (2*h)
```

- 数值扩散降为 O(u*h^2), 比一阶低 ~50x (ny=50)
- 需要 2 层 ghost cell (当前为 1 层, 需扩展)
- 边界处 (i=0, i=1) fallback 到一阶 upwind

数值扩散对比:

| Pe | 一阶 upwind | 二阶 upwind | 改善倍数 |
|----|------------|------------|---------|
| 10^3 | 2.5 | 0.05 | 50x |
| 10^5 | 250 | 5 | 50x |

Pe = 10^5 时二阶 upwind 的数值扩散仍为物理扩散的 5 倍. 可能还不够.

#### 1b. QUICK 格式 (三阶, 推荐)

```
u > 0 时:
adv = u * (-cc[i-2] + 6*cc[i-1] + 3*cc[i] - 2*cc[i+1]) / (6*h)
```

(注意系数因变体而异, 上面是一种常见形式)

实际常用形式 (对流项的 QUICK 插值):

```
face_flux = u * [ cc[i-1] + (3/8)*(cc[i]-cc[i-1])
                           + (3/8)*(cc[i-1]-cc[i-2])
                           - (1/8)*(cc[i]-2*cc[i-1]+cc[i-2]) ]
```

- 数值扩散 O(u*h^3), 对于 h=0.02 比一阶低 2500x
- Pe = 10^5 时数值扩散 ~ 0.1, 终于低于物理扩散
- 需要 2 层 ghost cell
- 可能引入小幅振荡 (不满足 TVD 条件)

#### 1c. TVD limiter (最稳健)

使用 flux limiter (Van Leer, Superbee, MC 等) 在二阶和一阶之间自适应切换:

```
phi(r) = (r + |r|) / (1 + |r|)        (Van Leer)
adv = u * [ (cc[i]-cc[i-1])/h + phi(r)/2 * (cc[i]-cc[i-1])/h * correction ]
```

- 光滑区域: 自动达到二阶精度
- 间断/sharp gradient 附近: 自动退化到一阶, 避免振荡
- 满足 TVD 条件, 保证 c >= 0 (物理有界性)
- 实现略复杂, 但代码改动仍在 advection_c 一个函数内

**推荐: 先实现 1b (QUICK), 如果出现振荡再加 limiter 升级到 1c.**

### Phase 2: 时间积分升级

**目标:** 放松 alpha 小、Pe 大时的 dt 约束.

在 Phase 1 完成后, CFL 约束仍然存在. 时间积分改进方案:

#### 2a. eta 半隐式 (已讨论, 最简单)

```
eta* = (eta_n + dt*eps*Da*c_n) / (1 + dt*eps*Da*(c_n + 1/K0))
```

- 去除 eta ODE 的稳定性约束
- 改一个函数 (calc_eta), 零额外成本

#### 2b. 对流 + 扩散隐式化

根据讨论, 方向 2 (GMRES/BiCGSTAB) 和方向 3 (ADI) 的取舍:

**方向 3 (ADI) 适用条件:** mesh Pe 数 < 2, 即 Pe*h < 2.
- 你的参数: Pe_max*h = 10^5 * 0.02 = 2000 >> 2
- **不适用**, Thomas 算法下一阶 upwind 的数值扩散问题会再次出现

**方向 2 (Krylov 迭代求解) 适用条件:** 任意 Pe, 但需要:
- 好的空间离散 (Phase 1 已解决)
- 好的预条件子 (ILU 或 AMG)
- 代码量较大

**方向 1 (Semi-Lagrangian) 重新评估:**
- 从根本上消除对流 CFL, 不受 mesh Pe 限制
- 插值扩散 O(h^4/dt) (双三次), 在 Pe = 10^5 下远小于物理扩散
- 扩散项仍可用 ADI 隐式 (此时 ADI 只处理扩散, 没有 mesh Pe 问题)
- 实现集中在替换 advection_c, 与 Phase 1 的改动位置相同
- **在你的参数范围下, 这可能是投入产出比最高的方案**

**推荐时间积分路线:**

```
2a (eta 半隐式) -> 2b-SL (Semi-Lagrangian 对流) -> 2b-ADI (扩散隐式)
```

每步独立可验证, 逐步解锁 dt.

### Phase 3: 高 alpha 区域优化 (可选)

alpha > 1 时, dt 受振荡精度约束 T_osc/20 = pi/(20*alpha^2*Sc).

对 alpha = 10, Sc = 16667: T_osc/20 ~ 10^(-8). 这是精度约束不是稳定性约束, 隐式化不能帮忙.

可能的优化:
- 自适应时间步进: 振荡流变化剧烈时用小 dt, 变化缓慢时用大 dt
- 多尺度方法: 对振荡流做时间平均, 用 effective dispersion 替代逐周期求解
- 这些是更深层的方法论改变, 建议在 Phase 2 完成并验证后再考虑

## 9. 风险与注意事项

1. **Ghost cell 扩展是破坏性改动.** Field2D 的内存布局、所有边界条件函数、checkpoint 格式都会受影响. 需要全面的回归测试.

2. **Semi-Lagrangian 在吸附区边界 (xpo_l, xpo_r) 附近需要特别处理.** 特征线可能从吸附区外回溯到吸附区内, 此处 eta 有跳变.

3. **CUDA 后端的 Semi-Lagrangian 实现.** 双三次插值的 texture memory 访问模式与当前 stencil 不同, 可能需要重写 kernel. 建议 CPU 版本验证正确后再移植.

4. **保持显式模式可用.** 建议通过 config 或命令行参数切换 explicit / imex, 不要删除现有显式代码, 方便 A/B 对比验证.
