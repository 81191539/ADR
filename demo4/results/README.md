# demo4 Results

Generated precheck, validation, and report files are written here.

Important outputs:

- `precheck/case_XXXX.json`: detailed theoretical diagnostics for one case.
- `precheck_summary.csv`: one-row-per-case diagnostic summary.
- `validation/case_XXXX/`: baseline refinement outputs and metrics.
- `demo4_summary.json`: combined summary for WebUI or later analysis.

The generated files are descriptive only. They list metrics and candidate
scheme rows without automatically selecting a best scheme.
