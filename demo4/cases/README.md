# demo4 Generated Cases

`validate_cases.py` writes per-case baseline refinement inputs here.

Generated content is organized as:

```text
demo4/cases/case_XXXX/
  baseline/input/input_parameter_XXXX.toml
  dt_refined/input/input_parameter_XXXX.toml
  ny_refined/input/input_parameter_XXXX.toml
  dt_ny_refined/input/input_parameter_XXXX.toml
```

These files are derived from existing case definitions and are safe to
regenerate.
